# 精读：arXiv 2510.02951 (Bot-Schindler 2025-10) 与 user Theorem 3 的 gap

## 一句话结论

**这篇论文不解决 user 的 Theorem 3 问题**，且 gap 是结构性的（连续时间 vs 离散时间、time-varying b(t) vs fixed-β、square-integrable diffusion vs constant variance i.i.d.、t→∞ asymptotic vs finite-horizon non-asymptotic）。**但其 Lyapunov 设计是可移植的关键思路**，特别是"twisted energy" `‖α(y - y*) + x‖²`，正是消除 momentum-position 跨耦合的 design 模板。

---

## 1. 论文究竟做了什么

### 1.1 模型：连续时间 SDE，不是离散迭代

`(SHBF)`：
```
dY(t) = X(t) dt
dX(t) = (-α X(t) - b(t) ∇f(Y(t))) dt + σ_Y(t) dW(t)
```
- α > 0：fixed friction（连续时间的 momentum 类比）
- b(t)：连续可微、**非减**的 stepsize-like 参数函数（关键变量）
- σ_Y(t) dW：Wiener 过程加性噪声，要求 ‖σ_Y‖_HS 平方可积
- t → ∞ 渐近分析（不是 finite horizon T）

### 1.2 主定理 (Theorem 2.2)

设 σ_Y 平方可积，则：
- (i) `f(Y(t)) - inf f = o(1/b(t))` a.s.
- (ii) `E[f(Y(t)) - inf f] = O(1/b(t))`
- (iii) Y(t) 弱收敛到 minimizer

### 1.3 改进定理 (Theorem 2.4) + Remark 2.5

在更强积分条件下，rate = O(1 / ∫_{t/2+t0}^t b(s) ds)。具体例子：
- `b(t) = t^r (r > 1)` → `f(Y(t)) - inf f = O(1/t^{r+1})`
- `b(t) = t^r log(t)` → `O(1/(t^{r+1} log t))`
- **`b(t) = const`** → 框架要求 b 非减且 sup{b'/b} < λ，常数 b 满足但 rate 仅为 `o(1/b)` = **O(1)，无衰减**
- `b(t) = exp(t)` → `O(1/exp(t))`

### 1.4 加速结果 (Theorem 2.6 / 2.7)

通过 time-rescaling，`(SHBF)` with specific `b(t) = (t-t0+s0/√(2(t-t0)))^...` 等价于 Su-Boyd-Candès 类系统 `(SAVD)`，给 `f(Z(s)) - inf f = O(1/s²)`。这是 Nesterov accelerated gradient 的连续时间版本，**不是 fixed-β SHB**。

### 1.5 假设

- f 是 convex + ∇f Lf-Lipschitz（即 L-smooth convex non-SC）✓ 与 user 一致
- 不需要 strong convexity ✓
- 不需要 quadratic growth ✓
- **noise**: σ_Y(t)dW（Wiener 过程），σ_Y 平方可积 ✗ 与 user 的 bounded variance i.i.d. 不一致

## 2. Gap 分析：四个结构性差异

| 维度 | 论文 (Bot-Schindler) | User Theorem 3 | Gap 性质 |
|---|---|---|---|
| **时间** | 连续时间 SDE | 离散迭代 y_{t+1} = y_t - η∇f + β(y_t - y_{t-1}) | 结构性，需要离散化误差分析 |
| **friction/momentum** | b(t) 非减、time-varying；b(t)=const 给 O(1) 无衰减 | β fixed ∈ (0,1)，η fixed | **致命**：固定 β 在连续时间对应 b=const，论文框架无法给 rate |
| **noise** | σ_Y(t)dW，σ_Y 平方可积 | i.i.d. mean-zero, E‖ξ_t‖² ≤ σ² | 论文要求 σ_Y(t) → 0，user 是 constant variance |
| **horizon** | t → ∞ 渐近 | finite T，η 调到 D/(σ√T) 与 T 关联 | 论文不给 horizon-tuned 的 explicit constant |

### 2.1 致命差异：fixed-β = 常数 b → 无 rate

论文 Remark 2.5 直接显示：要 rate `O(1/(t^{r+1}))`，需要 `b(t) = t^r`（增长）。**Constant b 给 O(1)，即只有渐近收敛、无 explicit rate**。这不是论文的疏漏，而是 fixed-friction HB 在连续时间确实没有比 ergodic 更好的 rate（Polyak-Alvarez 1964/2000 仅给 weak convergence）。

User 的 fixed-β SHB 在离散时间数值上有 T^{-0.50} —— 这只能来自 **离散时间的 finite-horizon + horizon-tuned η** 的特殊结构，不能从 t→∞ 连续时间分析得到。

### 2.2 致命差异：σ_Y 平方可积 vs constant variance

论文要求 ∫_{t0}^∞ ‖σ_Y(s)‖²_HS ds < ∞，即 **σ_Y 必须随时间衰减**。User 的 i.i.d. noise variance σ² 不衰减，离散-连续翻译后等价于常数 σ_Y，∫ σ²ds = σ²·t → ∞ 不可积。

→ 在 user 的 noise model 下，论文 Theorem 2.2 的"几乎处处收敛"结论本身就**不成立**（noise 累积发散，trajectory 不收敛到 minimizer）。这印证了 user 的 Theorem 3 要求 noise floor 项 `O(σD/√T)`（即 trajectory 不收敛到 0，而是 hover 在 σD/√T 量级附近）。

### 2.3 加速 → Su-Boyd-Candès 是 Nesterov，不是 fixed-β SHB

Theorem 2.6/2.7 的 O(1/s²) rate 是 Nesterov-style vanishing-damping system，对应 Nesterov accelerated gradient discretization。Fixed-β SHB 的离散化对应 fixed-friction HB，**不享有 O(1/s²) 加速**（这与 arXiv 2307.11291 的 "non-acceleration of HB" 结果一致）。

## 3. 可移植技术：Lyapunov 设计模板

虽然论文 setting 不直接 cover user 的问题，**Lyapunov 设计是 highly portable 的**。

### 3.1 核心 Lyapunov（论文 line 386–388）

```
E(t, y, x) = b(t)(f(y) - inf f) + (1/2)‖α(y - y*) + x‖² + (1/2)α(α-λ)‖y - y*‖²
```

三项的角色：
- **第一项** `b(t)(f - inf f)`：时间加权 function gap，Itô 展开后 b'(t)(f - f*) 给主衰减项
- **第二项** `‖α(y - y*) + x‖²/2`：**twisted energy**——把 momentum/velocity x 和 position offset y - y* 线性组合后取范数，这是处理 fixed-friction 的经典手法（Polyak 1964 原型）
- **第三项** `α(α-λ)‖y - y*‖²/2`：纯 position quadratic，参数 λ 调节谱半径

**关键观察**：第二项的 twist `α(y - y*) + x` 而非 `‖y - y*‖² + ‖x‖²` 单独，正是为了让 ⟨y - y*, x⟩ cross term 在求导后**自然消失**，这正是处理 momentum-position coupling 的标准技巧。

### 3.2 凸性的使用方式（line 421–423）

```
u(s) := b(s)(f(Y(s)) - inf f) - b(s)⟨∇f(Y(s)), Y(s) - y*⟩
     ≤ b'(s)(f(Y(s)) - inf f)·[symbol]
     ≥ 0   ← because of convexity: f(y) - f* ≤ ⟨∇f, y - y*⟩
```

凸性以 `f(y) - f* ≤ ⟨∇f(y), y - y*⟩` 形式进入，**没有用 smoothness**！只用了 convexity + Lipschitz gradient（用于存在性，不用于 rate）。这意味着如果离散版要 leverage smoothness，需要**额外**的 descent lemma 或 cocoercivity argument。

### 3.3 Itô / 鞅 + 噪声处理

`dE = drift dt + noise dW`：取期望或用 Doob 鞅不等式，`dW` 项消失，剩 `‖σ_Y‖²_HS dt`（quadratic variation）作为 noise floor。在 user 的离散 setting，等价于 `E[‖∇f‖² · η²]·σ²·T` 项作 floor。

## 4. 如何 adapt 到 user Theorem 3

如果要把论文技术 adapt 到 fixed-β + horizon-tuned η + finite T 离散 setting，需要做这几件事：

### 4.1 离散化论文的 Lyapunov

用论文的 `α = (1-β)/η`（连续时间 friction ↔ 离散 momentum 的常用映射），离散版 Lyapunov 候选：
```
V_t = w_t (f(y_t) - f*) + (1/2)‖α(y_t - x*) + (y_t - y_{t-1})/η‖² + (α(α-λ)/2)‖y_t - x*‖²
```
其中 w_t 是离散 b(t) 的类比，需要 design。**关键：fixed-β + finite T 下，w_t 应该不是 r 次多项式（那样发散），而可能是 `√t` 或常数或 √(T-t)**。这是 design challenge。

### 4.2 引入 smoothness

论文不用 smoothness（continuous setting 不需要）。离散版必须引入，候选：
- `f(y_{t+1}) ≤ f(y_t) + ⟨∇f(y_t), y_{t+1} - y_t⟩ + (L/2)‖y_{t+1} - y_t‖²`（descent lemma）
- `‖∇f(y_t)‖² ≤ 2L(f(y_t) - f*)` （smooth + convex 不等式）
- `⟨∇f(y) - ∇f(x), y - x⟩ ≥ (1/L)‖∇f(y) - ∇f(x)‖²` (cocoercivity)

这些是 Sanity Check 已经标记的"必须显式 leverage"的元素。

### 4.3 替换 noise 模型

把 σ_Y(t)dW 换成离散 i.i.d. ξ_t，对应：
- 鞅项 `Σ_{s≤t} ⟨[grad term], ξ_s⟩`：用 Doob 不等式或 conditional expectation
- 二次变差 `Σ_{s≤t} σ²η²`：累积成 σ²η² T 量级，匹配 user 期望的 noise floor σ²η T = σ² D √T / σ = σD√T（再除以 T 得 σD/√T，量级对）

### 4.4 Finite-horizon + horizon-tuned η

User 的 η = D/(σ√T) 是 T 依赖的。在 Lyapunov 求和到 t=T 后，η²T = D²/σ² (constant)，η/(1-β) = D/(σ(1-β)√T)。这给 `‖α(y - y*)‖²` 量级 `D²/(1-β)²`，与初始 ‖y_0 - x*‖² ≤ D² 量级匹配。

→ 论文的 t → ∞ 衰减视角不适用，必须 **sum to T、用 telescope sum 而非渐近极限**。

## 5. 净评估：这个方向值不值得继续投入？

### 5.1 论文不直接给答案

不。论文的 setting 与 user 问题 **结构性错配**：连续时间 vs 离散，time-varying b vs fixed-β，vanishing diffusion vs constant variance。即便完全消化论文，**也得不到 user Theorem 3 的 rate**。

### 5.2 但 Lyapunov 模板有价值

论文的 **twisted energy** `‖α(y - y*) + x‖²` 是一个明确的设计模板，提示离散版 Lyapunov 应该不是 user 之前试过的 `‖y_t - x*‖²` 单项 + `‖m_t‖²` 单项，而是 **‖α(y_t - x*) + (y_t - y_{t-1})/η‖²` 这种线性组合的范数**。这一点之前的 elementary routes (#3 直接 Lyapunov on (y_t, m_t)) 可能恰好没用对。

### 5.3 优先级修订

- **下调** #9 VI / monotone direction 的优先级：原本期望 arXiv 2510.02951 直接给 fixed-β 离散 SHB last-iterate，结果是连续时间 t→∞ 渐近，gap 大。
- **上调** #15 supermartingale-Lyapunov 的优先级：论文的 Itô 展开 + 鞅项 + quadratic variation 处理，正是离散 supermartingale 的连续时间镜像。把论文的 twisted energy 离散化 + 用 Robbins-Siegmund supermartingale + Doob 不等式 = 完整的 stochastic Lyapunov 框架。
- **保持** #7 iKM 优先级：与论文独立的另一条路。

### 5.4 推荐的下一步具体动作

1. **不要**继续花时间精读 arXiv 2510.02951 后续章节（monotone equation 部分对 user 问题不增量）。
2. **直接**用论文的 twisted Lyapunov 模板，构造离散版：
   ```
   V_t = w_t (f(y_t) - f*) + (1/2)‖α(y_t - x*) + (y_t - y_{t-1})/η‖² + (γ/2)‖y_t - x*‖²
   ```
   选 `α = (1-β)/η`，`w_t` 待定，`γ = α(α-λ)`。
3. 计算 V_{t+1} - V_t（用 SHB 迭代代入 + smoothness 不等式 + convexity 不等式）。
4. 对 stochastic noise，取条件期望 E[V_{t+1} | F_t] - V_t，证明 supermartingale 性质（即 ≤ -[positive] + σ²η²/(...)）。
5. Sum to T，得 E[V_T] - V_0 ≤ -Σ[positive] + σ²η²T/(...)；选 w_t 让 -Σ 项给 (f(y_T) - f*) 的累积。
6. 用 horizon-tuned η = D/(σ√T) 优化 constant。

这是一个明确的、有论文模板支持的、distinct from elementary routes 的攻击路线。
