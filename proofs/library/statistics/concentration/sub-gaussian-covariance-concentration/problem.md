# Sub-Gaussian Covariance Concentration

## Source
- Paper: Vershynin (2018/2020). "High-Dimensional Probability." Cambridge University Press, Ch. 4.

## Statement
For n×d matrix A with iid mean-zero sub-Gaussian rows (norm K, covariance Σ):
||(1/n)AᵀA - Σ|| ≤ K²max(δ,δ²), prob ≥ 1-2exp(-ct²), δ=C(√(d/n)+t/√n).
