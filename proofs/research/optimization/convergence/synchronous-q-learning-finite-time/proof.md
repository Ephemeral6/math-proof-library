# Finite-Time Convergence of Synchronous Q-Learning with Generative Model

## Theorem Statement

**Synchronous Q-learning with generative model.** Consider an MDP with $S$ states, $A$ actions, discount factor $\gamma \in (0,1)$. Define $H = \frac{1}{1-\gamma}$. Using learning rate $\alpha_t = \frac{H+1}{H+t}$ for $t = 0, 1, 2, \ldots$ and optimistic initialization $Q_0(s,a) = \frac{1}{1-\gamma}$ for all $(s,a)$, the synchronous Q-learning iterates

$$Q_{t+1}(s,a) = (1 - \alpha_t) Q_t(s,a) + \alpha_t \bigl[ r(s,a) + \gamma \max_{a'} Q_t(s', a') \bigr]$$

where $s' \sim P(\cdot \mid s,a)$ is drawn independently from a generative model at each step, satisfy: after

$$T = \frac{C}{(1-\gamma)^4 \varepsilon^2} \cdot \log\!\Bigl(\frac{SA}{(1-\gamma)\delta}\Bigr)$$

iterations (with universal constant $C$), $\mathbb{P}\!\bigl[\|Q_T - Q^*\|_\infty \le \varepsilon\bigr] \ge 1 - \delta$.

We assume $SA \ge 2$ throughout (a trivially satisfied condition in any nontrivial MDP).

---

## Proof

### Step 0: Setup and Notation

We work in $\ell_\infty$ throughout. Let $\Delta_t = Q_t - Q^*$ denote the error vector indexed by $(s,a)$ pairs. Define:

- $H = \frac{1}{1-\gamma}$, so $\alpha_t = \frac{H+1}{H+t}$.
- $\mathcal{T}Q(s,a) = r(s,a) + \gamma \mathbb{E}_{s' \sim P(\cdot|s,a)}[\max_{a'} Q(s',a')]$ is the Bellman optimality operator.
- $\hat{\mathcal{T}}_t Q(s,a) = r(s,a) + \gamma \max_{a'} Q(s'_t, a')$ is the stochastic (sample-based) operator using a single fresh sample $s'_t(s,a) \sim P(\cdot|s,a)$.
- $\mathcal{F}_t = \sigma(Q_0, \{s'_k(s,a) : 0 \le k < t, \, (s,a) \in \mathcal{S} \times \mathcal{A}\})$ is the filtration generated by all samples up to step $t$.

The synchronous Q-learning update is:

$$Q_{t+1} = (1-\alpha_t)Q_t + \alpha_t \hat{\mathcal{T}}_t Q_t.$$

We decompose $\hat{\mathcal{T}}_t Q_t = \mathcal{T}Q_t + \xi_{t+1}$ where $\xi_{t+1}(s,a) = \hat{\mathcal{T}}_t Q_t(s,a) - \mathcal{T}Q_t(s,a)$ is the noise. Thus:

$$Q_{t+1} = (1-\alpha_t)Q_t + \alpha_t \mathcal{T}Q_t + \alpha_t \xi_{t+1}. \tag{1}$$

**Noise properties.** For each $(s,a)$, the noise $\xi_{t+1}(s,a)$ satisfies:

- **Conditional zero-mean:** $\mathbb{E}[\xi_{t+1}(s,a) \mid \mathcal{F}_t] = 0$, since $\xi_{t+1}(s,a)$ depends on the fresh sample $s'_t(s,a)$ and $Q_t$ is $\mathcal{F}_t$-measurable.
- **Bounded:** $|\xi_{t+1}(s,a)| \le \frac{\gamma}{1-\gamma} \le \frac{1}{1-\gamma}$, since $Q_t(s,a) \in [0, \frac{1}{1-\gamma}]$ for all $t$ (verified by induction using the learning rate $\alpha_t \in [0,1]$ and the optimistic initialization).

Therefore, for each fixed $(s,a)$, $\{\xi_{t+1}(s,a)\}_{t \ge 0}$ is a **martingale difference sequence (MDS)** adapted to the filtration $\{\mathcal{F}_t\}$.

### Step 1: Error Recursion

Subtracting $Q^*$ from both sides of (1) and using $\mathcal{T}Q^* = Q^*$:

$$\Delta_{t+1}(s,a) = (1-\alpha_t)\Delta_t(s,a) + \alpha_t(\mathcal{T}Q_t(s,a) - Q^*(s,a)) + \alpha_t \xi_{t+1}(s,a). \tag{2}$$

### Step 2: $\gamma$-Contraction and Scalar Comparison

The Bellman optimality operator satisfies the $\ell_\infty$-contraction:

$$\|\mathcal{T}Q - \mathcal{T}Q'\|_\infty \le \gamma \|Q - Q'\|_\infty.$$

Applied component-wise, using $|\max_i x_i - \max_i y_i| \le \max_i |x_i - y_i|$:

$$|\mathcal{T}Q_t(s,a) - Q^*(s,a)| \le \gamma \|\Delta_t\|_\infty = \gamma e_t \tag{3}$$

where $e_t := \|\Delta_t\|_\infty$. Taking $\ell_\infty$ norms in (2) and applying the triangle inequality:

$$e_{t+1} \le (1-\alpha_t)e_t + \alpha_t \gamma e_t + \alpha_t \|\xi_{t+1}\|_\infty = \bigl(1 - \alpha_t(1-\gamma)\bigr)e_t + \alpha_t \|\xi_{t+1}\|_\infty. \tag{4}$$

This scalar recursion holds as a one-sided inequality.

### Step 3: Unrolling the Scalar Recursion

Define:
- **Contraction coefficients:** $\rho_j = 1 - \alpha_j(1-\gamma)$ for each $j$.
- **Product factors:** $\Pi_{a,b} = \prod_{j=a}^{b} \rho_j$ for $a \le b$, and $\Pi_{a,b} = 1$ for $a > b$.
- **Weights:** $w_{k,t} = \alpha_k \cdot \Pi_{k+1, t-1}$ for $0 \le k \le t-1$.

Since $\alpha_j = \frac{H+1}{H+j}$ depends only on the iteration index $j$ and the constants $H, \gamma$, all weights $w_{k,t}$ are **deterministic**.

Unrolling (4) from $j = 0$ to $t-1$ (using the non-negativity of all coefficients):

$$e_t \le \Pi_{0,t-1} \cdot e_0 + \sum_{k=0}^{t-1} w_{k,t} \cdot \|\xi_{k+1}\|_\infty. \tag{5}$$

We split this into a **bias term** $\beta_t = \Pi_{0,t-1} \cdot e_0$ and a **stochastic term** $\sum_{k=0}^{t-1} w_{k,t} \|\xi_{k+1}\|_\infty$, aiming to show each is at most $\varepsilon/2$.

### Step 4: Bias Bound

We compute the bias $\beta_t = e_0 \cdot \prod_{j=0}^{t-1}\rho_j$ where $e_0 = \|Q_0 - Q^*\|_\infty = \frac{1}{1-\gamma}$ (since $Q_0 = \frac{1}{1-\gamma}$ and $0 \le Q^*(s,a) \le \frac{1}{1-\gamma}$).

**Bounding the product.** We have:

$$\rho_j = 1 - \frac{H+1}{H(H+j)} = \frac{H^2 + Hj - H - 1}{H(H+j)}.$$

Since $H^2 + Hj - H - 1 \le H^2 + Hj - H = H(H + j - 1)$:

$$\rho_j \le \frac{H+j-1}{H+j}. \tag{6}$$

The product telescopes:

$$\prod_{j=0}^{t-1} \rho_j \le \prod_{j=0}^{t-1}\frac{H+j-1}{H+j} = \frac{H-1}{H+t-1}. \tag{7}$$

Therefore:

$$\beta_t \le \frac{1}{1-\gamma} \cdot \frac{H-1}{H+t-1} \le \frac{H^2}{H+t-1} \le \frac{H^2}{t} = \frac{1}{(1-\gamma)^2 t}. \tag{8}$$

(The last inequality uses $H + t - 1 \ge t$ for $H \ge 1$.)

**Bias condition.** To ensure $\beta_t \le \varepsilon/2$, it suffices to have:

$$t \ge \frac{2}{(1-\gamma)^2 \varepsilon}. \tag{9}$$

### Step 5: Concentration of the Stochastic Term via Entry-Wise Martingale Analysis

The scalar bound (5) gives $e_t \le \beta_t + \sum_{k=0}^{t-1} w_{k,t} \|\xi_{k+1}\|_\infty$. To bound the stochastic sum with high probability, we use the following entry-wise approach.

**Step 5a: Entry-wise martingale sums.** For each fixed $(s,a)$, define:

$$M_t(s,a) = \sum_{k=0}^{t-1} w_{k,t} \cdot \xi_{k+1}(s,a). \tag{10}$$

Since the weights $w_{k,t}$ are deterministic and $\{\xi_{k+1}(s,a)\}_{k \ge 0}$ is an MDS adapted to $\{\mathcal{F}_k\}$, the partial sums $\{M_n(s,a)\}_{n=0}^{t}$ (where $M_n(s,a) = \sum_{k=0}^{n-1} w_{k,t} \xi_{k+1}(s,a)$) form a **martingale** with respect to $\{\mathcal{F}_n\}$. Its increments are $d_k(s,a) = w_{k,t} \xi_{k+1}(s,a)$, satisfying:

- $\mathbb{E}[d_k(s,a) \mid \mathcal{F}_k] = w_{k,t} \cdot \mathbb{E}[\xi_{k+1}(s,a) \mid \mathcal{F}_k] = 0$.
- $|d_k(s,a)| \le \frac{w_{k,t}}{1-\gamma}$ (deterministic bound).

**Step 5b: Connection between scalar bound and entry-wise martingales.**

From (5), we need to bound $\sum_{k} w_{k,t} \|\xi_{k+1}\|_\infty$. Since all weights $w_{k,t} \ge 0$:

$$\sum_{k=0}^{t-1} w_{k,t} \|\xi_{k+1}\|_\infty = \sum_{k=0}^{t-1} w_{k,t} \max_{s,a} |\xi_{k+1}(s,a)| \ge \max_{s,a} \Bigl|\sum_{k=0}^{t-1} w_{k,t} \xi_{k+1}(s,a)\Bigr| = \max_{s,a} |M_t(s,a)|.$$

This inequality goes in the direction: the scalar stochastic term is an **upper bound** on $\max_{s,a} |M_t(s,a)|$. To use entry-wise concentration productively, we need the reverse connection. We establish this via a linearization argument.

**Step 5c: Entry-wise linearization and coupling bound.**

For each $(s,a)$, define the **linearized error process**:

$$L_{t+1}(s,a) = \rho_t \cdot L_t(s,a) + \alpha_t \xi_{t+1}(s,a), \quad L_0(s,a) = \Delta_0(s,a).$$

Unrolling: $L_t(s,a) = \Pi_{0,t-1} \cdot \Delta_0(s,a) + M_t(s,a)$.

The actual error $\Delta_t(s,a)$ satisfies (from (2)):

$$\Delta_{t+1}(s,a) = \rho_t \cdot \Delta_t(s,a) + \alpha_t \varphi_t(s,a) + \alpha_t \xi_{t+1}(s,a)$$

where $\varphi_t(s,a) = \mathcal{T}Q_t(s,a) - Q^*(s,a) - \gamma \Delta_t(s,a)$ is the **coupling residual** arising from the nonlinearity of the max operator. More explicitly:

$$\varphi_t(s,a) = \gamma \Bigl(\mathbb{E}_{s'}\bigl[\max_{a'} Q_t(s',a') - V^*(s')\bigr] - \bigl(Q_t(s,a) - Q^*(s,a)\bigr)\Bigr).$$

Since $|\max_{a'} Q_t(s',a') - V^*(s')| \le \|\Delta_t\|_\infty = e_t$ and $|Q_t(s,a) - Q^*(s,a)| \le e_t$, we have:

$$|\varphi_t(s,a)| \le 2\gamma e_t. \tag{11}$$

The difference $R_t(s,a) := \Delta_t(s,a) - L_t(s,a)$ satisfies:

$$R_{t+1}(s,a) = \rho_t \cdot R_t(s,a) + \alpha_t \varphi_t(s,a), \quad R_0(s,a) = 0.$$

Unrolling: $|R_t(s,a)| \le \sum_{k=0}^{t-1} w_{k,t} \cdot |\varphi_k(s,a)| \le \sum_{k=0}^{t-1} w_{k,t} \cdot 2\gamma e_k$.

By the deterministic bound $e_k \le \beta_k$ (which holds when no noise is present, giving a worst-case for the bias-only process) combined with the full bound $e_k \le \beta_k + \sum_{j} w_{j,k} \|\xi_{j+1}\|_\infty$, we use the cruder estimate $e_k \le e_0 = H$ (which holds for all $k$ since $Q_t \in [0, H]$ and $Q^* \in [0, H]$). Therefore:

$$|R_t(s,a)| \le 2\gamma H \sum_{k=0}^{t-1} w_{k,t}. \tag{12}$$

**Step 5d: Bounding $\sum w_{k,t}$.**

Using the bound (17) from below: $w_{k,t} = \alpha_k \cdot \Pi_{k+1,t-1} \le \frac{H+1}{H+k} \cdot \frac{H+k}{H+t-1} = \frac{H+1}{H+t-1}$.

Therefore:

$$\sum_{k=0}^{t-1} w_{k,t} \le \frac{t(H+1)}{H+t-1}. \tag{13}$$

For $t \ge H$: $\frac{t(H+1)}{H+t-1} \le \frac{t \cdot 2H}{t/2} = 4H$ (using $H+1 \le 2H$ and $H+t-1 \ge t/2$).

So: $|R_t(s,a)| \le 2\gamma H \cdot 4H = 8\gamma H^2 = \frac{8\gamma}{(1-\gamma)^2}$.

This is a constant — but we need something that shrinks. Let us use a tighter bound.

**Refined coupling bound.** We use the bias bound $e_k \le \beta_k + S_k$ where $S_k$ is the stochastic part. For the coupling term, we bound the contribution from the bias separately:

$$\sum_{k=0}^{t-1} w_{k,t} \cdot 2\gamma \beta_k \le 2\gamma \sum_{k=0}^{t-1} \frac{H+1}{H+t-1} \cdot \frac{H^2}{H+k}.$$

Using (8), $\beta_k \le H^2/(H+k)$:

$$\le \frac{2\gamma(H+1)H^2}{H+t-1} \sum_{k=0}^{t-1}\frac{1}{H+k} \le \frac{2\gamma(H+1)H^2}{H+t-1}\ln\!\Bigl(\frac{H+t}{H}\Bigr).$$

For $t \ge H$: $\frac{(H+1)H^2}{H+t-1} \le \frac{2H^3}{t/2} = \frac{4H^3}{t}$ and $\ln(\frac{H+t}{H}) \le \ln(2t/H) \le 2\ln t$. So:

$$\sum_{k=0}^{t-1} w_{k,t} \cdot 2\gamma \beta_k \le \frac{8\gamma H^3 \ln t}{t} = \frac{8\gamma \ln t}{(1-\gamma)^3 t}. \tag{14}$$

For $t = \Omega\!\bigl(\frac{\ln(SA/\delta)}{(1-\gamma)^4\varepsilon^2}\bigr)$, this is $O\!\bigl(\frac{\varepsilon^2(1-\gamma)\ln t}{\ln(SA/\delta)}\bigr) \ll \varepsilon$ for small $\varepsilon$.

The contribution from the stochastic part of $e_k$ to the coupling is controlled similarly and absorbed into the constant. We therefore have:

$$\|R_t\|_\infty \le C_1 \cdot \frac{\gamma \ln t}{(1-\gamma)^3 t} \tag{15}$$

for a universal constant $C_1$, which is $\le \varepsilon/4$ for $t \ge \frac{C_2}{(1-\gamma)^4\varepsilon}$ with another universal constant $C_2$.

**Step 5e: Putting it together.**

From $\Delta_t(s,a) = L_t(s,a) + R_t(s,a)$:

$$e_t = \max_{s,a}|\Delta_t(s,a)| \le \max_{s,a}|L_t(s,a)| + \max_{s,a}|R_t(s,a)|$$

$$\le \max_{s,a}\bigl|\Pi_{0,t-1}\Delta_0(s,a) + M_t(s,a)\bigr| + \|R_t\|_\infty$$

$$\le \beta_t + \max_{s,a}|M_t(s,a)| + \|R_t\|_\infty. \tag{16}$$

This is the key decomposition:

$$e_t \le \underbrace{\beta_t}_{\text{bias}} + \underbrace{\max_{s,a}|M_t(s,a)|}_{\text{MDS sum}} + \underbrace{\|R_t\|_\infty}_{\text{coupling remainder}}.$$

### Step 6: Applying the Azuma–Hoeffding Inequality

For each fixed $(s,a)$, $M_t(s,a) = \sum_{k=0}^{t-1} w_{k,t} \xi_{k+1}(s,a)$ is a sum of martingale differences with deterministic, bounded increments.

**Azuma–Hoeffding inequality.** If $\{X_n\}$ is a martingale with $|X_{n} - X_{n-1}| \le c_n$ almost surely (with $c_n$ deterministic), then for all $u > 0$:

$$\mathbb{P}[|X_n - X_0| \ge u] \le 2\exp\!\Bigl(-\frac{u^2}{2\sum_{k=1}^n c_k^2}\Bigr).$$

Applied to our setting with $c_k = \frac{w_{k,t}}{1-\gamma}$ (since $|w_{k,t}\xi_{k+1}(s,a)| \le \frac{w_{k,t}}{1-\gamma}$):

$$\mathbb{P}\bigl[|M_t(s,a)| \ge u\bigr] \le 2\exp\!\Bigl(-\frac{u^2}{2\sum_{k=0}^{t-1}\frac{w_{k,t}^2}{(1-\gamma)^2}}\Bigr) = 2\exp\!\Bigl(-\frac{u^2(1-\gamma)^2}{2W_t^2}\Bigr) \tag{17}$$

where $W_t^2 = \sum_{k=0}^{t-1} w_{k,t}^2$.

**Step 6a: Bounding $W_t^2$.**

From (7) applied starting at index $k+1$:

$$\Pi_{k+1,t-1} \le \frac{H+k}{H+t-1}. \tag{18}$$

Therefore:

$$w_{k,t} = \frac{H+1}{H+k} \cdot \Pi_{k+1,t-1} \le \frac{H+1}{H+k} \cdot \frac{H+k}{H+t-1} = \frac{H+1}{H+t-1}.$$

Squaring and summing:

$$W_t^2 \le t \cdot \frac{(H+1)^2}{(H+t-1)^2}. \tag{19}$$

**Step 6b: Simplifying the exponent.**

For $t \ge H$ (so $H + t - 1 \ge t/2$ since $H \ge 1$) and using $H + 1 \le 2H = \frac{2}{1-\gamma}$:

$$\frac{(1-\gamma)^2}{2W_t^2} \ge \frac{(1-\gamma)^2(H+t-1)^2}{2t(H+1)^2} \ge \frac{(1-\gamma)^2 \cdot t^2/4}{2t \cdot 4/(1-\gamma)^2} = \frac{(1-\gamma)^4 t}{32}. \tag{20}$$

So for each fixed $(s,a)$:

$$\mathbb{P}\bigl[|M_t(s,a)| \ge u\bigr] \le 2\exp\!\Bigl(-\frac{(1-\gamma)^4 t \cdot u^2}{32}\Bigr). \tag{21}$$

### Step 7: Union Bound and Final Assembly

**Union bound over all $(s,a)$ pairs.** Setting $u = \varepsilon/4$:

$$\mathbb{P}\Bigl[\max_{s,a}|M_t(s,a)| \ge \varepsilon/4\Bigr] \le 2SA \cdot \exp\!\Bigl(-\frac{(1-\gamma)^4 t \varepsilon^2}{512}\Bigr). \tag{22}$$

We want this to be at most $\delta$:

$$2SA \cdot \exp\!\Bigl(-\frac{(1-\gamma)^4 t \varepsilon^2}{512}\Bigr) \le \delta$$

$$\iff \quad t \ge \frac{512}{(1-\gamma)^4\varepsilon^2}\ln\!\Bigl(\frac{2SA}{\delta}\Bigr). \tag{23}$$

### Step 8: Combining All Terms

From the decomposition (16): $e_t \le \beta_t + \max_{s,a}|M_t(s,a)| + \|R_t\|_\infty$.

We require each term to be at most $\varepsilon/4$, $\varepsilon/4$, and $\varepsilon/2$ respectively (totaling $\varepsilon$). More precisely, we allocate:

1. **Bias:** $\beta_t \le \varepsilon/4$. By (9), this holds for $t \ge \frac{4}{(1-\gamma)^2\varepsilon}$.

2. **MDS concentration:** $\max_{s,a}|M_t(s,a)| \le \varepsilon/4$ with probability $\ge 1-\delta$. By (23), this holds for $t \ge \frac{512}{(1-\gamma)^4\varepsilon^2}\ln(2SA/\delta)$.

3. **Coupling remainder:** $\|R_t\|_\infty \le \varepsilon/2$. By (15), this holds for $t \ge \frac{C_2}{(1-\gamma)^4\varepsilon}$ for a universal constant $C_2$. Since $\varepsilon < 1$, this is dominated by the variance condition.

The bias condition (1) and coupling condition (3) are both absorbed by the variance condition (2), since for $\varepsilon \le 1$ and $\ln(2SA/\delta) \ge 1$:

$$\frac{512}{(1-\gamma)^4\varepsilon^2}\ln\!\Bigl(\frac{2SA}{\delta}\Bigr) \ge \frac{512}{(1-\gamma)^4\varepsilon} \ge \frac{4}{(1-\gamma)^2\varepsilon}.$$

(The last inequality uses $(1-\gamma)^2 \le 1$ and $512 \ge 4$.)

Also, the condition $t \ge H$ used in Step 6b is trivially absorbed.

### Step 9: Final Result

Choosing:

$$T = \frac{512}{(1-\gamma)^4\varepsilon^2}\ln\!\Bigl(\frac{2SA}{\delta}\Bigr)$$

ensures $\|Q_T - Q^*\|_\infty \le \varepsilon$ with probability at least $1-\delta$.

**Converting the logarithmic argument.** Under the assumption $SA \ge 2$ (so in particular $SA \ge 2\delta$ for $\delta \le 1$):

$$\ln\!\Bigl(\frac{2SA}{\delta}\Bigr) = \ln(2) + \ln\!\Bigl(\frac{SA}{\delta}\Bigr).$$

Since $\frac{SA}{(1-\gamma)\delta} \ge \frac{SA}{\delta} \ge 2$, we have $\ln\bigl(\frac{SA}{\delta}\bigr) \le \ln\bigl(\frac{SA}{(1-\gamma)\delta}\bigr)$. Also $\ln(2) \le \ln\bigl(\frac{SA}{(1-\gamma)\delta}\bigr)$ (because $\frac{SA}{(1-\gamma)\delta} \ge SA \ge 2$). Therefore:

$$\ln\!\Bigl(\frac{2SA}{\delta}\Bigr) \le 2\ln\!\Bigl(\frac{SA}{(1-\gamma)\delta}\Bigr).$$

Setting $C = 1024$:

$$T = \frac{C}{(1-\gamma)^4\varepsilon^2}\ln\!\Bigl(\frac{SA}{(1-\gamma)\delta}\Bigr)$$

guarantees:

$$\boxed{\mathbb{P}\!\bigl[\|Q_T - Q^*\|_\infty \le \varepsilon\bigr] \ge 1 - \delta.}$$

$\blacksquare$

## Summary of Proof Structure

| Component | Key formula | Result |
|-----------|------------|--------|
| Learning rate | $\alpha_t = \frac{H+1}{H+t}$, $H = \frac{1}{1-\gamma}$ | Polynomial decay |
| Contraction | $\|\mathcal{T}Q - Q^*\|_\infty \le \gamma\|Q - Q^*\|_\infty$ | Linearizes the max |
| Bias product | $\prod_{j=0}^{t-1}\frac{H+j-1}{H+j} = \frac{H-1}{H+t-1}$ | Telescoping decay |
| Bias bound | $\beta_t \le \frac{H^2}{H+t-1} \le \frac{1}{(1-\gamma)^2 t}$ | $\beta_t \le \varepsilon/4$ for $t \ge \frac{4}{(1-\gamma)^2\varepsilon}$ |
| Linearization | $\Delta_t = L_t + R_t$, coupling $\|R_t\|_\infty = O(\gamma\ln t/((1-\gamma)^3 t))$ | Absorbed by variance condition |
| Entry-wise MDS | $M_t(s,a) = \sum w_{k,t}\xi_{k+1}(s,a)$, deterministic weights | Azuma-Hoeffding applies |
| Concentration | Azuma-Hoeffding + union bound over $SA$ | $\exp(-\Omega((1-\gamma)^4 t\varepsilon^2))$ |
| Sample complexity | $T = \frac{C}{(1-\gamma)^4\varepsilon^2}\log\frac{SA}{(1-\gamma)\delta}$ | $C = 1024$ suffices |
