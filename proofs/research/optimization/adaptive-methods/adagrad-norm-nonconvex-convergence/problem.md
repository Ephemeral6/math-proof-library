# AdaGrad-Norm Non-Convex Convergence Rate

## Source
- Paper: Ward, Wu, Bottou, "AdaGrad stepsizes: sharp convergence over nonconvex landscapes", ICML 2019 / JMLR 2020.
- Related: Li & Orabona 2019 "On the Convergence of Stochastic Gradient Descent with Adaptive Stepsizes" (AISTATS).
- Context: AdaGrad-Norm is the scalar-stepsize variant of AdaGrad. This was the first proof that a purely adaptive step size (no knowledge of $L$ or $\sigma$) achieves $\tilde{O}(1/\sqrt{T})$ non-convex convergence without projection or clipping.

## Statement

**Setup.** Let $f: \mathbb{R}^d \to \mathbb{R}$ be $L$-smooth, i.e.
$$\|\nabla f(x) - \nabla f(y)\| \le L \|x - y\| \quad \forall x, y \in \mathbb{R}^d,$$
and assume $f^\star := \inf_x f(x) > -\infty$. Let $\Delta_0 := f(x_0) - f^\star$.

**Algorithm (AdaGrad-Norm).** Fix $\eta > 0$ and $b_0 > 0$. For $k = 0, 1, 2, \dots$ define
$$b_k^2 \;=\; b_0^2 + \sum_{i=0}^{k-1} \|g_i\|^2, \qquad x_{k+1} \;=\; x_k - \frac{\eta}{b_k}\, g_k,$$
where $g_k$ is a stochastic gradient oracle queried at $x_k$, with $\mathcal{F}_k := \sigma(g_0, \dots, g_{k-1})$.

**Assumptions.**
- **(A1) Unbiasedness.** $\mathbb{E}[g_k \mid \mathcal{F}_k] = \nabla f(x_k)$.
- **(A2) Bounded noise.** $\|g_k - \nabla f(x_k)\| \le \sigma$ almost surely for some $\sigma \ge 0$.
- **(A3) Bounded true gradient.** $\|\nabla f(x_k)\| \le G$ almost surely along the trajectory, for some $G \ge 0$.

(Assumption A3 is a standard simplifying assumption used in Ward-Wu-Bottou 2019 Theorem 2.1. A2 + A3 imply $\|g_k\| \le G + \sigma$ a.s.)

**Claim.** There exists a constant $C = C(L, \eta, b_0, \sigma, G, \Delta_0)$ — polynomial in its arguments — such that for all $T \ge 2$,
$$\mathbb{E}\!\left[\min_{0 \le k < T} \|\nabla f(x_k)\|^2\right] \;\le\; C \cdot \frac{\log T}{\sqrt{T}}.$$

Equivalently, the expected average gradient norm satisfies
$$\frac{1}{T} \sum_{k=0}^{T-1} \mathbb{E}\!\left[\|\nabla f(x_k)\|^2\right] \;=\; O\!\left(\frac{\log T}{\sqrt{T}}\right).$$

## Difficulty

**research** — the proof requires a non-trivial combination of:
1. Adaptive descent lemma with a non-predictable step size $\eta/b_k$ (correlated with noise).
2. A logarithmic integral-test bound on $\sum_k \|g_k\|^2 / b_{k+1}^2$.
3. Careful handling of the conditional cross-term $\mathbb{E}[\langle \nabla f(x_k), g_k \rangle / b_k]$ — $b_k$ is $\mathcal{F}_k$-measurable but step sizes involving $b_{k+1}$ are not.
4. Combining a deterministic "potential-style" bound with an expectation argument to avoid needing $b_T$ to be controlled w.h.p.

## Key intermediate results to prove

1. **Adaptive descent lemma.** For all $k$,
 $$f(x_{k+1}) \;\le\; f(x_k) - \frac{\eta}{b_k} \langle \nabla f(x_k), g_k \rangle + \frac{L \eta^2}{2 b_k^2} \|g_k\|^2.$$

2. **Logarithmic accumulator bound.** For any positive sequence $a_0, \dots, a_{T-1}$ with $B_0 > 0$ and $B_k^2 = B_0^2 + \sum_{i<k} a_i^2$,
 $$\sum_{k=0}^{T-1} \frac{a_k^2}{B_{k+1}^2} \;\le\; \log\!\left(\frac{B_T^2}{B_0^2}\right).$$
 (This is a discrete version of $\int dx/x = \log$.)

3. **Decoupling lemma (key technical step).** Show that one can replace $1/b_k$ by $1/b_{k+1}$ in the stochastic cross-term with controllable error:
 $$\mathbb{E}\!\left[\sum_{k=0}^{T-1} \frac{\langle \nabla f(x_k), g_k - \nabla f(x_k) \rangle}{b_k}\right] \;\text{is bounded by}\; O(1) + \text{error of order } O(\log T).$$
 One standard approach: use the identity $1/b_k = 1/b_{k+1} + (b_{k+1} - b_k)/(b_k b_{k+1})$ and bound the correction.

4. **Envelope for $b_T$.** $b_T^2 \le b_0^2 + T (G + \sigma)^2$, so $b_T \le b_0 + (G+\sigma)\sqrt{T}$.

5. **Final assembly.** Summing the descent lemma over $k = 0, \dots, T-1$, rearranging, and combining (2)-(4) gives
 $$\mathbb{E}\!\left[\sum_{k=0}^{T-1} \frac{\|\nabla f(x_k)\|^2}{b_k}\right] \;\le\; \frac{\Delta_0}{\eta} + O(\log T).$$
 Multiplying both sides by the bound on $b_T$ in (4) and using $b_k \le b_T$ gives the claimed $O(\log T / \sqrt{T})$.

## Notation summary

- $\|\cdot\|$ denotes the Euclidean 2-norm throughout.
- $\mathbb{E}[\cdot]$ is expectation w.r.t. all randomness (oracle noise across all iterations).
- $\mathcal{F}_k$ is the $\sigma$-algebra generated by $g_0, \dots, g_{k-1}$. Note $x_k, b_k \in \mathcal{F}_k$, but $g_k \notin \mathcal{F}_k$.
- $\log$ denotes natural logarithm.
