# Condition R00 — Relation only (discrete_curvature)

## 数据分析

我有 summary_delta + per-vertex 数据：
- `vertex_defects_changed`: 每次变换后哪些顶点的角缺改变了
- `new_vertex_defects`: 新增顶点的角缺（stellar 的中心点 ≈ 0）
- `sum_of_vertex_changes`: 受影响顶点上 Σ(post-pre) 角缺
- `total_curvature_pre/post/_preserved`: 全部 100 case 都是 12.566... = 4π
- `curvature_types_pre/post`: 例如 tetrahedron 4 个 positive 顶点

观察：
1. **Σδ 在 100 个 case 上恒等于 4π = 2π × 2**（χ_pre = χ_post = 2）。
2. **对 stellar subdivision**: 原 face 三个角点的 defect 不变（splitting flat triangle in plane 不动 corner angles），新的中心点 defect = 0（共面的 3 段贡献回了 2π 的总角度）。整体 Σδ 不变。
3. **对 vertex displacement**: 一个顶点 + 它的 3 个 neighbors 的 defect 都改变了，但 `sum_of_vertex_changes = 0` —— 局部曲率重分布、全局曲率保持。
4. **每个 preset 顶点角缺值是 π / (2π/3) / (π/3) ...** 都是 π 的有理数倍。

## 论证尝试

我看到：
- 每个顶点 v 的 defect = 2π - Σ(angles around v)
- 多面体顶点处所有 angles 的和 = 2π - δ(v)
- 我能 *验证* Σδ = 4π 但没有把它和 χ 的代数式联系起来的工具。

从 displacement 的 "sum_of_vertex_changes = 0" 我能推断：变换是局部重分布、不改变全局 — 但这只是 **invariance**，不是 G-B 等式本身。

要证 Σδ = 2πχ，我需要：
- 对每个三角形：内角和 = π → 求和得到 πF
- 同一总和按顶点重组：= Σ_v (2π - δ(v)) = 2πV - Σδ
- 因此 πF = 2πV - Σδ
- 配合 3F = 2E（闭曲面）得到 χ = V - F/2，从而 Σδ = 2πχ

**问题**：R 数据没给我每个三角形的角度，也没明确告诉我 3F = 2E。我只能 hand-wave 而非论证。

## 自评

**评分：PATTERN（2）**

理由：我看到 Σδ = 2πχ 的稳定 pattern + 局部重分布的机制证据，但缺少"每个面贡献 π"的关键步骤来推导等式。如果我假定那一步，我可以拼出完整证明，但 R 数据本身没支撑那一步。
