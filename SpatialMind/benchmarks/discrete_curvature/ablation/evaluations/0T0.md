# Condition 0T0 — Transform only (discrete_curvature)

## 数据分析

我有 summary_delta + transform_trace：
- `before_state` / `after_state`: V, E, F, χ, Σδ, gauss_bonnet_ratio
- `delta`:
  - stellar: vertices_added=1, edges_added=3, faces_added=2, euler_change=0, curvature_change=0
  - displacement: euler_change=0, curvature_change=0, local_defect_change=±x
- `region_affected` (stellar):
  - subdivided_face / new_vertex_index
  - **`local_defect_sum_before` = `local_defect_sum_after`**：被分割面三个角点的 defect 总和不变
  - `new_vertex_defect` ≈ 0 —— 加在面中心的新点是 flat
- `region_affected` (displacement):
  - displaced_vertex / neighbor_faces / affected_vertices

## 论证尝试

T 数据给我两件事：
1. **Stellar 的纯组合恒等式**: ΔV - ΔE + ΔF = 1 - 3 + 2 = 0 — 这说明 χ 是任何"加点 + 完整三角剖分"操作的不变量，因为每加一个内点会带 +1 V、+3 E、+2 F。
2. **局部曲率守恒**: 在被改动的 region 内，defect_sum 守恒（stellar 是 0 + flat new vertex；displacement 是 single vertex change but ... 注意：displacement 的 trace 只显示一个顶点的 defect change，没显示邻居的，所以从 T 数据看不到"邻居改变也加和为 0"的细节）。

不过从 T 数据出发要构造 Σδ = 2πχ 的证明，缺失：
- 没有每个面的角度信息 → 无法用 πF
- 没有每个顶点的全局 defect 表 → 只能看到局部受影响的部分
- 没有反例去验证"closedness 是必要前提"

从 `before_state.gauss_bonnet_ratio = 2.0` 横跨所有 case 我能 *观察* 等式，但 *推导* 它需要整体几何而非 local trace。

## 自评

**评分：PATTERN（2）**

理由：T 数据让我清楚看到 χ 的组合不变性 + 局部曲率守恒，但缺少"每个三角形角和 = π"和"3F=2E"的全局信息，无法形成本地→全局的证明链。
