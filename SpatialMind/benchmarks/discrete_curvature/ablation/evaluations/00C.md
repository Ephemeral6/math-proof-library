# Condition 00C — Contrastive only (discrete_curvature)

## 数据分析

summary_delta（同 000）+ 3 个 critical CF：

**CF1 boundary_relaxation**: 完整 stellar (ΔV=1,ΔE=3,ΔF=2) → 退化拼接 (ΔV=1,ΔE=2,ΔF=1)。
- delta_chi = 0（仍然守恒）
- delta_curv = +6.283 = +2π
- gauss_bonnet_violated = True
- explanation 关键：**新顶点周围 0 度角 → defect = 2π 全部贡献到 Σδ**。Σδ 跳了 +2π，χ 没动 → G-B 失效。

**CF2 operation_perturbation**: 删一个面（保留边）。
- delta_chi = -1
- delta_curv = +π
- gauss_bonnet_violated = True
- explanation: 删去一面，3 个角点各损失一个角，**3 个角加起来 = π** → Σδ 增加 π。同时 ΔF=-1 → Δχ=-1 → 2πχ 减 2π。两边方向相反，差 3π。

**CF3 condition_removal**: 让一个面引用越界顶点 → 程序报错。说明 manifold 一致性是前提条件。

## 论证尝试

CF1 和 CF2 暴露了两个关键机制：
1. **Defect 的"角缺"含义**: 新顶点周围 0 度 → defect = 2π。即 defect 衡量的是"应该是 2π 但少了多少"。
2. **每个三角形的角和 = π** (从 CF2 推断): 删一个面让 Σδ 增加 π — 即原本"被吃掉"的 π 角度回到了角缺账户。

把 (1)(2) 拼起来：每个三角形对全局角和贡献 π；每个顶点周围有 (2π - δ(v)) 角度。所以
   π · F = Σ_v (2π - δ(v)) = 2πV - Σδ  ⟹  Σδ = 2πV - πF.

但是没有 R 数据我无法独立验证这个等式（我看不到具体顶点的 defect 值），也没有 V/E/F 的具体值（summary_delta 只给 delta）。所以我只能"假定"等式成立、"猜测"3F=2E（闭曲面），从而推到 Σδ = 2πχ。

## 自评

理由：CF 给了我两条机制线索（defect = 2π 表示完全角缺；删面 → Σδ 增 π 表示 face contributes π），这是论证的关键骨架。但没有 R 数据来确认每个 mesh 的具体 V、defect 分布，也没有 T 数据来观察 ΔV/ΔE/ΔF 的纯组合关系，所以我只能 *猜* 出证明而非 *构造* 它。
