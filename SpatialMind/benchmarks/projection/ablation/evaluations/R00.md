## 数据分析
summary 之外多了：

**detailed_comparison**：
- `collisions_introduced`：投影后有多少对点重合
- `crossings_introduced`：投影后有多少对边交叉
- `fraction_distances_preserved`：投影后保持原距离的点对比例
- `diameter_change`：直径变化值（≤ 0）

每对的 detailed level 还包含 `distance_matrix_a / _b`（每个对象的距离矩阵）。

**structural_comparison**：
- `distances_preserved_count / distances_total`：保持/总数
- `dimension_before` = 3, `dimension_after` = 2
- `information_loss_summary`：{collisions, crossings, distance_distortion}
- `invariants_preserved`：{n_points, dimension, diameter} 各自的 True/False
- `invariants_delta`

我的关键观察：

**(O1) n_points 恒守恒**（invariants_preserved.n_points = True 在 66/66 case 上）。

**(O2) dimension 必然丢失**（invariants_preserved.dimension = False 在 66/66 case 上）。

**(O3) diameter 守恒条件**（来自 invariants_preserved.diameter）：  
- tetrahedron 的 4 个投影：diameter 守恒在 4/4。  
- octahedron：xy/xz/yz 守恒 (3/3)，diagonal 不守恒。  
- triangular_prism：xz 守恒，xy/yz/diagonal 不守恒（其中 diagonal 部分情况守恒）。  
- cube：所有 4 个投影 diameter 都不守恒。  
- square_antiprism：仅 diagonal 守恒。

**结论**：diameter 守恒 ⟺「3D 对象的最长对角线在投影平面内有非零分量」。例如 tetrahedron 的对角线方向恰在每个坐标平面内（顶点是 (±1,±1,±1) 形式），所以所有投影都保持。cube 的最长对角线是 (1,1,1) 方向，对每个坐标平面来说有压缩。

**(O4) fraction_distances_preserved**：所有 66 个 case 都 < 0.5（最高 0.50，tetrahedron diagonal）。最低 0.07（triangular_prism diagonal、square_antiprism diagonal 部分）。绝大多数在 0.3–0.45 范围。  

**结论**：投影 *普遍* 破坏大多数距离对。即使最对称的对象也只能保持约一半。

**(O5) collisions 与对象对称性的关系**：
- tetrahedron：所有 4 个投影都 0 collision（顶点几何不对称地排布，没有点会重合）。
- cube xy/xz/yz：4 collisions（每个面的 4 个顶点和对面的 4 个顶点重合）。
- cube diagonal：1 collision（(0,0,0) 和 (1,1,1) 都映到 (0,0)）。
- octahedron：恰好 1 collision in xy/xz/yz（顶点 (0,0,±1) 等之一）。
- square_antiprism xz/yz：3 collisions。

**结论**：collisions 数 = #{投影方向上排成同一直线的点对}。对称性高的对象有更多 collision。

**(O6) distance_matrix_a 与 distance_matrix_b 比较**（来自 detailed level）：  
- self-projection: a = obj, b = obj (3D)，dm_a = dm_b（恒等）；变换后 dm_a' 是 2D 距离，与 dm_b 比较显示损失。  
- cross_object：a = objA (3D), b = objB (3D)，dm_a ≠ dm_b（两个不同对象有不同的距离矩阵）。投影后 dm_a' 是 2D，需要看 dm_a' 是否能等于 dm_b 来判断撞影。  
  从数据看 dm_a'(cube_xy) ≠ dm_b(square_antiprism_3D)，所以这两个对象在 xy 投影下没有撞影。但 dm_a' 与 dm_b 的差异不是 0 不是 trivial pattern——distance matrix 大致相似（都有相同的 8 个点 + 类似的边集），但具体数值不同。

## 论证尝试

**问题 (a)：哪些三维性质在所有投影下不变？**

从 (O1)、(O2)：  
- `n_points` 在 66/66 守恒；  
- `dimension` 在 66/66 不守恒；  
所以 n_points 是 *全局守恒量*。dimension 必丢。

`diameter` 不是全局守恒（来自 (O3)）。但守恒条件是「最长对角线 *有* 投影平面内分量」。  

**完整论证**：设 3D 对象 X 的直径由点对 (p, q) 实现。投影 π 把 (p, q) 映到 (π(p), π(q))。对正交投影，‖π(p) − π(q)‖² = ‖p − q‖² − ‖p_⊥ − q_⊥‖²，其中 p_⊥, q_⊥ 是丢掉那个轴的分量。所以：
- ‖p_⊥ − q_⊥‖ = 0（直径方向 *完全* 在投影平面内）→ 直径完全守恒；
- ‖p_⊥ − q_⊥‖ > 0 → 直径减少（甚至直径 *变短* 也意味着另一个点对成为新直径，但 monotone：2D diameter ≤ 3D diameter 严格）。  

数据 (O3) 的所有「守恒」case 都满足前者；所有「不守恒」case 都满足后者。

**问题 (b)：不同对象能否撞影？**

逻辑回答：能。我可以构造证据：取两个 3D 对象 A 和 B，使得它们在某投影 π 下产生相同的 2D 点集。例如：A = 单位立方体，B = 立方体 + 任意延伸到 z 方向的额外点（点的 z 分量任意）。然后 π_xy(A) = π_xy(B) — 显然 *很多* 不同的 3D 对象映到同一 2D 投影。这是「投影非单射」的标准事实。

但是，**这个事实在 R 数据中并没有体现**——cross_object case 比较的是不同对象的 *自身* 投影，而不是构造「撞影 case」。R 数据可以告诉我「cube xy 投影 ≠ square_antiprism xy 投影」（distance matrix 不同），但不能告诉我「不存在两个不同对象的投影相同」（这是更强的 universal claim）。

**问题 (c)：能否从投影重建？**

数据告诉我 `fraction_distances_preserved < 0.5` 恒成立 (O4)。所以 2D 距离矩阵 *不能* 唯一编码 3D 距离矩阵——至少一半的距离信息丢了。**重建不唯一**。

但这只能说「2D 距离矩阵不足以恢复 3D 距离矩阵」。要严格证明「不可恢复」，需要构造两个 3D 对象映到同一 2D 距离矩阵的反例，这在 R 数据里没有给出（cross_object 已经是 distance matrix 不同的对象）。

## 自评
**评分：ARGUMENT（3）**

理由：
- (a) 给出了 `n_points` 恒守恒、`diameter` 守恒条件的完整代数论证。这条 *论证*。  
- (b) 逻辑上能回答（投影非单射），但 R 数据里没有「两个对象撞影」的具体 case，所以是 *外部知识 + 数据 inference*，不是 *从数据论证*。  
- (c) 只能用 (O4) `frac < 0.5` 推「信息不够」，但严格不可重建需要反例（数据里没有）。  
- 整体论证链不闭合到 PROOF：(a) 闭合，(b)(c) 部分依赖外部反例。
