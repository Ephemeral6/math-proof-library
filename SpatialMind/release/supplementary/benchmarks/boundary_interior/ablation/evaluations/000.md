# Condition 000 — no R, no T, no C (boundary_interior)

## 数据分析

87 个 (P, P') pair 的 `summary_delta`，每个字段是 pre 与 post 之差（或 `0` 表示 unchanged，或 `{"pre":..,"post":..}` 表示具体变化）。

观察：
- 大多数 case 的 `summary_delta` 几乎全部为 0（translate / shear / add_vertex / identity_compare 都不改变 area, B, I, pick_holds）。
- `scale_non_uniform` case（约 16 个）的 `area_a, B_a, I_a` 都是非零的正整数 delta。
- `pick_holds_a, pick_holds_b` 全部 = 0（恒为 True；从未变化）。

`metadata.operation` 揭示每个 case 的操作类型（translate / shear / add_vertex / scale_non_uniform / identity_compare），但**没有任何 pre/post 的具体数值**——只能看到 delta，不能看到 A、B、I 各自是多少。

## 论证尝试

**我能观察的结构事实**：
- 4 类操作中有 3 类（translate, shear, add_vertex）保持 (A, B, I) 三元组完全不变。
- 1 类（scale_non_uniform）以正比方式改变三者（具体 ratio 不知，因为没有 pre 值参考）。
- pick_holds 在所有 case 上都从 True 保持到 True。

**我猜测的命题**：
- "lattice 多边形上 A 与 (B, I) 可以联立，因此从 (B, I) 能反推 A" — 但我看不到 (A, B, I) 的具体值，只看到它们是否变化，所以无法验证任何具体公式（如 Pick 的 A = I + B/2 − 1）。
- "scale_non_uniform 让面积放大某个 ratio，B 和 I 也按某种规律放大" — 没数据支撑具体规律。

**我的核心局限**：summary_delta 是*差分*视图，缺少绝对量纲。Pick 是*等式*命题，需要 (A, B, I) 的具体值才能验证或推导。

## 自评

**评分：PATTERN−（1）**

理由：
- 观察到了正确的结构事实（多数变换保持 invariants），但
- 看不到 (A, B, I) 的具体值，无法验证或推导 Pick 公式；
- pick_holds 恒为 True 这一信号被压缩成 "delta = 0"，看不到这是因为 Pick 在 lattice 简单多边形上恒成立。
- 因此连"目标命题在数据上成立"都无法明确陈述（只能说 pick_holds 不变化），更别说证明。
