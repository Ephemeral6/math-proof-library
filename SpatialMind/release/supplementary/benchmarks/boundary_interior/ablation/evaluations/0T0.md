## 论证尝试

**经验定理（从 87 case 提取）**：对所有测试的 lattice 简单多边形，A = I + B/2 − 1。

**变换分类（trace 给出）**：
- translate, shear (det=1), add_vertex: 三类操作都让 (A, B, I) 完全不变。
  - translate 显然保 A、B、I（平移不改变面积、不改变边界格点拓扑）。
  - shear 整数 unimodular（det=±1）保面积（线性代数事实），且把整数格双射到自身（保 B、I 的计数）。
  - add_vertex 在已有边的内部格点添加：边上的格点本就被 B 计入；加一个顶点把一条边切成两条短边，但短边的 gcd 之和 = 原边 gcd（每条短边对应原边上的一段格点链），所以 B、A、I 都不变。
- scale_non_uniform [sx, sy] 把 A 乘以 sx·sy（线性变换的 det），B 和 I 也以 Pick 兼容方式改变（因为变换后仍是 lattice 多边形）。

**Pick 公式的论证**：
- (i) trace 给出 87 个独立的 (A, B, I) 三元组，全部满足 A = I + B/2 − 1。这是强经验证据。
- (ii) trace 中的 add_vertex 测试展示了 B 计数方式的稳定性：在边上加格点不改变 B，因为 B 只数"位于边界几何上的格点"，不数顶点本身。
- (iii) 但 trace 不暴露多边形顶点坐标和边的几何，所以不能从原理上推导 Pick——只能验证它持续成立。

**为什么变换分类 ↔ Pick 不变性**：
- 保 A 的变换（translate/shear/add_vertex）也保 B 和 I（数据观察）；
- 改 A 的变换（scale_non_uniform）按比例改 B 和 I，使 Pick 仍成立（数据观察）；
- 这暗示 Pick 不是"巧合"，而是 lattice 多边形的*结构等式*——任何让多边形仍是 lattice 简单的变换都自动让 Pick 继续成立。

理由：

- 因此达到 ARGUMENT（条件 → 结论的推理链 + 经验验证），但 PROOF 需要 R 提供的边界几何或外部数学知识填补。
