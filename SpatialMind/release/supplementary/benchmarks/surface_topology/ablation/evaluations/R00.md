# Condition R00 — R only

## 数据分析
1563 个 case 的关键 R 信号：

**Bigon 结构**（structural_comparison）
- `bigons_without_puncture_pre / _post = 0` **for all 1563 cases**（pre AND post）。
- `bigons_with_puncture` 等于 `bigons_total`（pre 范围 1–12, post 范围 0–12）。
- 因此 `all_bigons_contain_puncture` 恒为真（modulo 0-bigon vacuous case：29 个 case 的 post = 0，标为 false）。
- `minimal_position_pre / _post = false` 恒成立（α 与 β 在三角剖分中并非紧 / 最小位置）。
- 每个 bigon 的 `tip_punctures` 含 1 或 2 个 puncture。

**候选交叉点**（detailed_comparison）
- `crossings_outside_region_pre = crossings_outside_region_post = 0` **for all 1563**——所有候选交叉点都在变换区域内。
- `crossings_pre_count` 范围 3–16（mean 11.91），`crossings_post_count` 范围 2–16（mean 8.74）。常常下降。
- `crossings_in_region_pre = crossings_pre_count`、`crossings_in_region_post = crossings_post_count`（与第一条等价）。
- δ(crossings_pre, post) 的直方图：527 个 case 不变，236 个减 2，178 个减 4，等等。
- 候选交叉点变化与 i(σ_α, β) − i(α, β) 没有简单的代数关系（如 `in_region − 2*bigons` 等于 i 这种猜测，只在 55–138/1563 个 case 上成立）。

## 论证尝试
观察构成的 hypothesis：
> α 与 β 上的所有 bigon 都包含至少一个 puncture，且所有候选交叉点都落在 surgery 区域里。

这暗示：
- bigon 上有 puncture ⟹ bigon 是「essential」的（puncture 阻止了 isotopy 把 bigon 缩掉），所以候选 bigon 不是 minimal-position 偏差的来源。
- candidate crossings 的下降不是「移除可缩 bigon」造成的，因为没有可缩 bigon。

但我无法把这个 hypothesis 闭合成一个论证：
- 我不知道 surgery 怎么作用（我没有 T 数据）。
- 我不知道 i(α, β) ≤ 1 的角色（我没有 C 数据；在所有 case 上 i = 1，所以 `i ≤ 1` 是普遍假设，但我无法判断它是否 critical）。
- candidate crossings 经常下降、bigon 也跟着下降，但 intersection number 不变——我说不清这种「冗余」消去的机制。

无法构造证明。

## 自评
**评分：PATTERN（2）**

理由：观察到了一个普遍且非平凡的结构 pattern——「每个 bigon 都含 puncture」+「所有候选交叉都在变换区域内」。这是对机制的 *structural observation*，但缺论证步骤（不知道 surgery 怎么操作，也不知道哪条假设是 load-bearing）。
