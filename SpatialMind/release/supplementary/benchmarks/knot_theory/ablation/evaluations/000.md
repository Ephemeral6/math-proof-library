# Condition 000 — no R, no T, no C (knot_theory)

## 数据分析
100 个 (K, K') pair 的 `summary_delta`：
- `crossing_count_a`: 恒 = +2（R2 添加 2 个交叉）
- `writhe_a` / `writhe_b` / `writhe_delta`: 恒 = 0
- `linking_number`: 恒 = 0（单分量纽结）

`metadata` 给出每个 pair 的 knot 名（10 种）、trial 索引、pre/post 的 writhe 和 crossing 数。

## 论证尝试
我看到的结构事实：
- **R2 add +2 crossings**（操作定义，不是不变量）
- **writhe 恒守恒**
- **linking number 恒守恒**

但是 prompt 要证明的是 **signature, determinant, Alexander polynomial** 守恒——这三个量在 summary 里**完全没有**。所以我连"目标定理在数据上成立"都验证不了，更别说机制。

我能做的只是：
- 观察到 writhe 和 linking number 这两个 peripheral invariant 在所有 100 case 上守恒。
- 推测："如果 R2 保持 writhe，也许它保持其他 sign-related invariants" — 但这只是猜测，没有数据支持。

## 自评
**评分：PATTERN−（1）**

理由：观察到 writhe + linking number 守恒，是 *real conservation laws*，但这两个不是目标不变量。无法验证 signature/det/Alex 是否守恒，更无法解释为什么。
