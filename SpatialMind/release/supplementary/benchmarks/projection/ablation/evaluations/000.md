# Condition 000 — no R, no T, no C (projection)

## 数据分析
66 个 case 的 `summary_delta`，加上 `metadata`：
- `n_points_a/_b`：恒为 0（前后点数不变，每对都是 0 delta）
- `dimension_a` = -1（3 → 2），`dimension_b` 视 case 而定（self/cross_object: 0；cross_projection: -1）
- `diameter_a` 通常 ≤ 0（投影后直径不增加）
- `diameter_b` = 0（参照对象 b 不变换）
- `edge_crossings_a`：投影后才有意义，pre=None / post 有数值

`metadata` 给出：case_type ∈ {self_projection, cross_projection, cross_object}、对象名、平面、n_points、n_edges。

我从 summary 看到的事实：
- **n_points 恒不变**（投影不扔点）。即使有「重合」也只是 collision，不是 deletion。
- **dimension 减 1**（200/200 case），是投影的定义。
- **diameter 一般减少**（66/66 个 case 中 diameter_a ≤ 0），有些 case = 0（直径方向恰好在投影平面内，未被压缩）。
- **edge_crossings_a** 有正数 = 投影后边交叉。

我看不到的：
- 哪些点对的距离被保持、哪些被破坏；
- 哪些点重合了；
- 哪些三维对象在投影下变成相同 2D 形状；
- 反事实数据。

## 论证尝试

我能从 summary 间接观察「diameter_a 的 0 和负值的分布」：tetrahedron 的所有 4 个投影 diameter_a = 0（即 3D 直径 = 2D 直径），但 cube 的 3 个轴向投影 diameter_a ≈ -0.32（√3 → √2）。这暗示「直径方向是否在投影平面内」决定 diameter 守恒，但 summary 没有给出方向信息，我只能看到「有 / 没有 diameter loss」。

对题目要求的三个问题：

(a) 哪些三维性质在所有投影下都不变？  
- 从 summary 我能看到 `n_points` 恒不变（66/66）— 这是个守恒律。  
- `dimension` 必然减 1（不是守恒）。  
- `diameter` 在某些 case 守恒，在其他 case 不守恒——summary 给不出哪个 case 守恒的判据。  

(b) 不同对象能否撞影？  
- summary 看不到点的具体位置，无法判断两个不同的 3D 对象在投影下是否产生相同的 2D 点集。

(c) 能否从投影重建？  
- summary 看不到 distance matrix。无法回答。

## 自评
**评分：PATTERN（2）**

理由：观察到「n_points 恒守恒」和「dimension 必然减 1」两条平凡事实，加上 diameter 损失分布暗示某种方向依赖。但没有距离数据、没有 collisions 数据、没有反事实，无法触及任何核心机制——投影的信息损失原因、撞影、重建——这些都需要更深的字段。
