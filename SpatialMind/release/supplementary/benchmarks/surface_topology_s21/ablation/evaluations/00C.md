# Condition 00C — Contrastive only

## 数据分析

**summary_delta 分布（128 cases，全部 i(α,β)=1）**：
抽样统计：intersection_number ∈ {0, -1}，weight_b 恒为 0，weight_a 恒为负且按 α 分组（a122: -14；a151: -9；a215: -22；a221: -28；a249: -22）。
按 i_change 划分：约 92/128 case 的 intersection_number = 0（surgery 保持），约 36/128 case 的 intersection_number = -1（surgery *降低*相交数 1）。提示中宣称的 92/36 split 与数据一致。

**关键观察（仅靠 summary_delta 可见）**：
- net_change=-1 的 case 在每个 α 内都出现，并不集中于某一个 α；说明 failure 不是某条特定 α 的退化。
- 在 a122 块里，beta_level=1 的 case 大量出现 i_change=-1（如 b2,b6,b9,b13,b17,b24,b45,b57,b63,b65,b124,b177），而 beta_level=0 的 case（如 a151 多数）大量保持 i_change=0。这是数据中**一个真实的、可见的二分**：beta_level 似乎与 net_change 部分相关（虽不完美，a151-b6/13/55/57/63/150 等 beta_level=1 的 case 也是 -1，与之一致；但 a215-b13/17/34/44/76 beta_level=1 却是 0，所以并非严格判据）。

**counterfactual 块（3 个 CF）**：
1. *boundary_relaxation*: i≤1 → i=2，net_change 从 -1 → -2，delta=-1。`condition_is_critical=false`。说明放松边界让失效**变得更糟**——i=2 上信号更负，但因为 i=1 的正例池本来就有 -1，所以"边界保不变性"这件事没有形成"PROCEED vs FAIL"的强对比，只是从"已经 -1"到"-2"，是 degree 加重，不是 phase transition。
2. *condition_removal*: 去掉 descending-link 约束，net_change 从 -1 → 0，delta=+1。这才是**唯一一个产生反向信号**的 CF：去掉一个**额外的**结构条件后失败反而消失，提示 i ≤ 1 不足，descending-link/level 对比才是真正决定性的隐藏条件。
3. *operation_perturbation*: 换 surgery curve（a_0 → c0），delta=0；说明 surgery curve 选择本身不是关键。

## 论证尝试

i(α,β) ≤ 1 在 S_{2,1} 上**不足**保 surgery 不变性——36/128 ≈ 28% 的正例池本身已失效，CF1 (boundary relaxation) 也只是 -1→-2 的渐变，没法把 i≤1 与 i=2 划成 PROCEED/FAIL 两类。
真正的对比信号来自 **CF2**：当且仅当再附加 "β 在 descending link 中、level(β) < level(α)" 时，net_change 才被推到 -1；去掉这个条件 net_change=0。这正是 a122 块里 beta_level=1 case 倾向于 -1 而 beta_level=0 case 倾向于 0 的 summary_delta 模式背后的解释。
因此可猜测加强后的命题：**若 i(α,β) ≤ 1 且 β 不在 α 的 descending link（或 level(β) ≥ level(α)），则 i(σ_α,β)=i(α,β)**。这与 summary_delta 的部分二分模式吻合，也与 CF2 的反向 delta 一致；CF1 仅给出弱的"放松后更差"signal。
但仍有反例（a215-b13/17 beta_level=1 却 i_change=0；a221-b13/70/78 beta_level=1 是 -1），说明 level 对比只是必要的近似条件，不充分。

## 自评

**评分：PATTERN (2)**
理由：CF2 (condition_removal) 给出了一个真正的反向 delta（-1 → 0），把 i ≤ 1 失败的 36 个 case 归因到一个**可被 summary_delta 部分验证**的隐藏结构（descending-link/level 对比）。这超出 NO_SIGNAL，也比 WRONG_PATTERN 强：模式存在并被两块独立证据（CF + summary_delta 中 a122 的 beta_level 二分）相互佐证。
但相较 S_{1,2} 的 PATTERN：(a) 正例池本身 28% 失效，使 CF1 boundary 信号被稀释成"程度差异"而非"相变"，discriminative power 明显下降；(b) summary_delta 中的 level 模式有反例（a215, a221），不能升到 ARGUMENT；(c) 没有 relation/transform 原语，无法验证 descending-link 的几何含义。综合判定为 PATTERN 偏弱端，不降到 WRONG_PATTERN，因为信号方向正确且 CF2 的 delta 不是噪声。
