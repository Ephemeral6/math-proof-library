# Condition 000 — only summary

## 数据分析

- 共 128 个 case，全部 i_alpha_beta = 1，weight_b 全为 0（β 不参与 surgery，符合设定）。
- intersection_number delta 分布：92 个为 0（保持），36 个为 -1（违反等式，σ_α 与 β 的交点比 α 与 β 少 1）。违反率 36/128 ≈ 28%。
- weight_a 完全由 alpha_index 决定，5 个 α 类：a122(weight_a=-14, 27 cases)、a151(-9, 43)、a215(-22, 13)、a221(-28, 19)、a249(-22, 26)。weight_a 数值与违反率无单调关系。
- 按 alpha_level 分桶：level=2 → 18/70 违反 (26%)；level=3 → 12/45 (27%)；level=4 → 6/13 (46%)。alpha 越复杂，违反率略升，但样本不平衡。
- **关键交叉表（intersection × beta_level）**：
  - beta_level=0：6/67 违反 (9%)
  - beta_level=1：24/49 违反 (49%)
  - beta_level=2：6/10 违反 (60%)
  - beta_level=3：0/2 违反 (0%)
- beta_level=0 与 beta_level≥1 之间的违反率有 5× 差距（9% vs 47%），p 值显然显著。

## 论证尝试

数据透露的结构：β 的 "level"（推测为 β 在 train track 上的复杂度，或与 surgery region 相对的位置层级）是 surgery invariance 失败的主要预测变量。最自然的解读：

- beta_level=0 → β 大致"远离"或"轻微触及"surgery region，σ_α 替换不破坏交叉模式。
- beta_level≥1 → β 深度穿过 region，canonical sigma 在 S_{2,1} 上不是 universal filler，导致 i(σ_α, β) = 0 而非 1。

因此仅靠 i(α,β) ≤ 1 在 S_{2,1} 上**不充分**；需要附加条件，例如"β 与 surgery region 的几何相交受限"，可由 beta_level=0 近似刻画。但 beta_level=0 仍有 6/67 反例（9%），说明该条件还不是充要条件——仍缺少能区分这 6 个边缘 case 的细粒度特征（在 summary-only 数据中已不可达）。

无法构造严格证明：summary_delta 不含 region 内/外的具体位置、不含 σ_α 的 train track 表示、不含 i(α,β) 的实际值（仅元数据 i_alpha_beta=1，无 surgery 后值，只有 delta）。所有论证都停留在统计相关层面。

## 自评

**评分：PATTERN (2)**

理由：仅凭 summary 数据，发现了一个 5× 强度的预测变量（beta_level），它把违反率从 9% 提升到 47-60%，定性地支持"β 与 surgery region 的位置关系决定 invariance"的假设——这是 prompt 提示的核心区分能力。但 (a) beta_level=0 仍有 9% 例外，所以未达到充要条件；(b) 没有几何/拓扑层面的可验证论证，只是 metadata 上的关联。够 PATTERN，不够 ARGUMENT。
