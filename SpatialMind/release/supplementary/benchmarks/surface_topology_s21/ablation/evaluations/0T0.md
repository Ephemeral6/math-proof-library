# Condition 0T0 — Transform only

## 数据分析

128 cases，net_change=0 共 92，net_change=-1 共 36（baseline bad-rate 28.1%）。

**全局 T 常量（在 ok / bad 两组都一致）：**
- `region.triangles` 大小恒为 3（即 region 占曲面 3/6 = 一半），所有 128 case。
- `long_arc_triangles` 恒为空（[]），所以"长弧"维度在 T 数据里是退化的。
- `punctures_in_region` 恒为 1。
- `short_arc_triangles` 大小取值 ∈ {4,6}：ok 中 55:37（6:4），bad 中 30:6。

**单变量 ok / bad 分布（[ok, bad]）：**
- `beta_triangles_total`: 6→[85,27]=24%bad, 3→[6,7]=54%bad, 2→[1,2]=67%bad
- `beta_triangles_in_region`: 3→[85,31], 2→[0,2], 0→[7,3]
- `level_shift`: -2→[72,30], -3→[20,6]
- `weight_shift`: -14→[15,12]=44%, -9→[37,6]=14%, -22→[27,12]=31%, -28→[13,6]=32%
- `beta_through_short_arc`: 6→[55,23], 4→[30,4], 1→[6,0], 3→[0,7], 2→[0,2], 0→[1,0]
- `beta_through_long_arc`: 恒为 0。

**关键的判别测试：把 9 个 T 特征拼成元组**（tri大小, short大小, punct, level_shift, weight_shift, β_tot, β_in, β_short, β_long），128 case 仅落入 **14 个唯一桶**：
- 7 个纯 bad 桶 → 共 9 个 bad case
- 2 个纯 ok 桶 → 共 7 个 ok case
- **5 个混合桶 → 装着 85 个 ok + 27 个 bad**（占 ok 的 92%、bad 的 75%）

也就是说，**T 特征对绝大多数 case 不能区分 net_change=0 / -1**。最大的混合桶 (3,6,1,-2,-14,6,3,6,0) 是 ok=15 / bad=10，bad-rate 40% — 完全无法判定。

部分单变量信号确实存在：β_short ∈ {3,2} → 9 个全 bad；β_short=1 → 6 个全 ok；weight_shift=-9 的 bad-rate 14%（baseline 28%）。这些都是 *边缘* 桶，覆盖率小。

## 论证尝试

prompt 的目标问题：i(α,β) ≤ 1 是否足以保 i(σ_α,β) = i(α,β)？

数据直接回答：**不**。36/128 反例存在，且 prompt 已声明 i(α,β) ≤ 1（metadata.i_alpha_beta 在抽样的 case 里就是 1）。

下一个目标：能否从 T 数据刻画"额外条件"使得在该条件下不变性恒成立？

- **β 落在 region 内 vs 外**：β_in_region=0（即 β 完全在 region 外）有 7 ok / 3 bad — 即使 β 完全外面也有 30% 失效；不是充分条件。
- **β 通过 short arc 的次数**：β_short=1 → 6 个全 ok（小但纯的子集），但仅 6 case，不可推广。
- **(weight_shift, β_short) 联合**：weight_shift=-9 ∧ β_short=4 → ok=30, bad=4（11.8% bad）— 比 baseline 好但仍非零。
- prompt 强调的"区分 region 内/外"在 0T0 数据里几乎是退化的：bad 中 27/36 都是 β 经过全部 6 个三角形（即 β_in=3、β_out=3），ok 中 85/92 也是同样模式。这两组在最关键的"β 与 region 关系"上**几乎完全重合**。

所以 T 数据虽然在 S_{2,1} 上表面看起来"非退化"（region≠surface），但用于区分 surgery 不变性是否成立时仍**不充分**：long_arc_triangles 恒空、region 大小恒为 3、punctures_in_region 恒为 1，这些都不携带变化信息；可变的只有 short_arc 大小、level/weight_shift、β_total/β_in/β_short，而这些组合产生的 14 桶里一大半混合。

诚实结论：**没有从 T 数据里抽出一个判别 36 反例的充分条件。**只观察到弱相关（β_total<6、β_short∈{2,3} 倾向于 bad；β_short=1、weight_shift=-9 倾向于 ok），这是 pattern 级的，不构成 argument。

## 自评
**评分：PATTERN (1)**

理由：
- 不是 NO_SIGNAL：T 特征确实有非平凡分布（不像 S_{1,2} 上恒定），且能在小子集上 100% 分类（7 ok 全在 2 个纯 ok 桶，9 bad 全在 7 个纯 bad 桶）。weight_shift、β_short 等单变量呈现真实的 ok/bad 倾斜（14% vs 28% vs 44%）。
- 但远未达 ARGUMENT：5 个混合桶吞下 85 ok + 27 bad，占两组绝大多数；T 特征**不能判定**典型 case。prompt 提示的"region 内 vs 外"这一应当最具区分力的轴在数据里几乎退化（绝大多数 case β 全部在 region 内或全部经过 6 三角形）。
- 没有构造性证明，没有充分条件刻画，只有边缘桶上的统计倾向。给 PATTERN(1) 是诚实的。
