# Condition 000 — no R, no T, no C (symmetry_combinatorics)

## 数据分析
200 个 (a, b) pair 的 `summary_delta`：
- `hamming_distance`: 变换后 σ_a 与 b 之间的 hamming 距离差
- `same_orbit`: 0 或 "unchanged"（恒为 0/unchanged）
- `orbit_size_a` / `orbit_size_b`: 都为 0（因为 σ_a 和 a 同轨道，所以 orbit_size_a 不变）

`metadata` 给出：标签 (same/diff)、g_idx、g_perm、a/b 的 coloring、orbit_size_a/b、n_distinct_a/b。

我从数据看到的结构事实：
- **same_orbit_post == same_orbit_pre** 在 200/200 上成立 — 群作用保持 same_orbit 关系。
- 100 个标签为 `same` 的 case 上 same_orbit = True；100 个 `diff` 上 same_orbit = False。
- 所有 case 的 orbit_size 在变换后不变。

## 论证尝试
我能看到一个真实的守恒律：群元素作用保持轨道。但这条几乎是定义层面的——σ_a = g·a 显然与 a 同轨道，所以与 b 的同轨道关系自然由 a 与 b 的关系决定。

prompt 要求证明的三个目标：
1. same_orbit 是等价关系；
2. |Stab(a)| × |Orbit(a)| = |G|（轨道-稳定子）；
3. 轨道数 = (1/|G|) Σ |Fix(g)|（Burnside）。

我在 summary 里**完全看不到** stabilizer、Fix(g)、轨道总数。我连「轨道数到底是多少」都不知道。从 metadata 只能看到 orbit_size_a ∈ {1, 2, 3, 6}（推测自 200 个样本的取值），但这不足以得出 |Stab|×|Orbit|=|G|，因为我没有 |Stab| 数据。

唯一稍有意义的观察：orbit_size 的取值都是 6 的因子（1, 2, 3, 6），暗示某种 Lagrange 式约束，但这只是模式识别。

## 自评
**评分：PATTERN（2）**

理由：观察到「群作用保持 same_orbit」（这是接近定义的事实）和「orbit_size ∈ 6 的因子集」（一个真实但非平凡的数论模式），但完全没有 stabilizer、Fix(g) 数据，无法触及 orbit-stabilizer 定理或 Burnside 引理本身。
